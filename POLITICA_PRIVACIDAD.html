2. 📊 Información que Recopilamos

2.1 Datos Técnicos de Funcionamiento

La aplicación procesa localmente en tu dispositivo:

  • 🔌Información de dispositivos USB: VID/PID, fabricante, modelo y estados de conexión del programador.
  • 📄Registros de operaciones: Logs de comandos ejecutados y resultados de lectura/escritura/verificación.
  • 💾Archivos binarios: Firmware seleccionado por el usuario para operaciones de memoria.
⚠️ Importante: Todos estos datos se procesan únicamente en tu dispositivo. No se envían a servidores externos ni se almacenan fuera de tu control.

2.2 Servicios de Terceros (Futuras Integraciones)

El proyecto contempla integrar servicios de terceros para:

  • 📱Google AdMob: Publicidad personalizada (puede incluir identificadores publicitarios cuando aplique).
  • 📊Microsoft App Center: Análisis de uso, métricas de rendimiento y reportes de estabilidad.

Si estos servicios son activados en futuras versiones, se actualizará esta política y se te notificará a través de la actualización de la aplicación.

3. 🔑 Permisos Solicitados

La aplicación requiere los siguientes permisos para funcionar correctamente:

  • 🔌Acceso a USB (HOST): Necesario para comunicarse con programadores USB vía OTG.
  • 📂Acceso a almacenamiento (SAF): Para seleccionar, guardar y exportar archivos de firmware mediante el sistema de documentos de Android.
  • 🌐Acceso a Internet: Para futuras funcionalidades en línea, servicios de analítica y publicidad (cuando aplique).

4. 🎯 Uso de la Información

La información recopilada se utiliza exclusivamente para:

  • ✅ Ejecutar operaciones de lectura, escritura, verificación y respaldo de chips de memoria.
  • ✅ Mostrar resultados, logs y mensajes de diagnóstico en tiempo real.
  • ✅ Mejorar la estabilidad, rendimiento y experiencia de usuario de la aplicación.
  • ✅ Permitir la importación y exportación de archivos binarios de firmware.

5. 💾 Almacenamiento y Control de Datos

Tú tienes el control total sobre los archivos y datos que maneja la aplicación:

  • Todos los archivos de firmware son gestionados por ti mediante el selector de archivos de Android.
  • Los logs y registros se almacenan temporalmente en memoria RAM y se limpian al cerrar la app.
  • Puedes borrar todos los datos de la aplicación en cualquier momento desde Ajustes → Aplicaciones → Flash EEPROM Tool → Borrar datos.

6. 🛡️ Seguridad de los Datos

Implementamos medidas de seguridad para proteger tu información:

  • 🔒 Procesamiento local de todos los archivos (sin envío a servidores).
  • 🔒 Sin almacenamiento de datos sensibles en la nube.
  • 🔒 Permisos mínimos necesarios para el funcionamiento.
  • 🔒 Código fuente abierto disponible en GitHub para auditoría transparente.

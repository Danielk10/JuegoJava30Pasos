2. Datos que puede recopilar la aplicación

La aplicación puede procesar datos técnicos de funcionamiento para diagnóstico, tales como:

  • Información de conexión USB (por ejemplo VID/PID y estados de conexión).
  • Registros técnicos (logs) de ejecución de comandos.
  • Archivos binarios seleccionados por el usuario para lectura/escritura/verificación.

Además, el proyecto contempla integrar servicios de terceros para monetización y analítica:

  • Google AdMob: para publicidad (incluyendo identificadores publicitarios cuando aplique).
  • App Center: para analíticas de uso y estabilidad.

3. Permisos

La app puede requerir permisos vinculados a:

  • Acceso a dispositivos USB OTG para comunicación con programadores.
  • Selección y guardado de archivos mediante el sistema de documentos de Android.
  • Acceso a Internet para funcionalidades en línea y futuras integraciones de servicios.

4. Uso de la información

La información se utiliza para:

  • Ejecutar funciones de lectura, respaldo, verificación y escritura de firmware.
  • Mostrar resultados y mensajes de diagnóstico al usuario.
  • Mejorar estabilidad y experiencia del producto.

6. Conservación y control

El usuario controla los archivos que importa/exporta. Puede borrar datos de la app desde ajustes de Android.

7. Cambios a esta política

Esta política puede actualizarse para reflejar cambios técnicos, legales o de servicios integrados.

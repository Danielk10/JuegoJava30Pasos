Supported programmers¶

flashrom supports many different programmers, for the full list you can look into programmer_table.c in the source tree. Some of the programmers have their own documentation pages, see below.

Please check the PROGRAMMER-SPECIFIC INFORMATION section of the Manual page for more details about programmers and their usage.

If you can run flashrom locally, the command flashrom -L prints all devices supported per programmer (see Manual page for more details on command line options). The output of this command is long, so you might want to save it to file or grep.

Patches to add/update documentation are very much appreciated.

  • CH341A/B
  • Dummyflasher
  • FT2232 SPI

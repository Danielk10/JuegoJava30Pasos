🔒 Política de Privacidad

Flash EEPROM Tool

📅 Última actualización: 2 de marzo de 2026

En Flash EEPROM Tool, respetamos tu privacidad y nos comprometemos a proteger tus datos personales. Esta política detalla cómo se recopilan, utilizan, comparten y protegen los datos de los usuarios, así como el uso de identificadores publicitarios y permisos necesarios para el correcto funcionamiento de la app.

1. 📊 Recopilación, Uso y Compartición de Datos

1.1 Servicios de Google AdMob (Publicidad)

Esta aplicación utiliza Google AdMob para mostrar anuncios. Este es el único servicio de terceros con el que se comparte información. Para ello, se recopila y utiliza:

  • 🆔ID de Publicidad de Android: Un identificador único (AAID) que permite a Google personalizar anuncios y realizar análisis de fraude.
  • 📈Métricas de Interacción: Datos sobre impresiones de anuncios y clics para optimizar el servicio publicitario.

1.2 Datos Técnicos Locales (No compartidos)

La aplicación procesa estrictamente de forma local en tu dispositivo:

  • 🔌Información de hardware USB: VID/PID y modelo del programador conectado.
  • 💾Archivos binarios: El firmware que selecciones para leer o escribir en tus memorias EEPROM. Estos archivos nunca salen de tu dispositivo ni se transmiten a ningún servidor externo.
Nota sobre AdMob: Puedes restablecer o limitar el uso de tu ID de publicidad desde los ajustes de Google en tu dispositivo Android.

2. 🔑 Permisos Solicitados

Para cumplir con sus funciones técnicas y publicitarias, la app requiere:

  • 🔌Acceso a USB (HOST): Para la comunicación directa con programadores vía OTG.
  • 📂Almacenamiento (SAF): Para importar y exportar tus archivos de firmware de forma segura.
  • 🌐Acceso a Internet: Necesario para la carga de anuncios de AdMob y reporte de errores.
  • 📶Bluetooth: Requerido por las librerías de AdMob y para la detección de periféricos de programación inalámbricos compatibles.
  • 📱Estado de Red: Para verificar la conectividad antes de intentar cargar servicios externos.

3. 🛡️ Retención y Eliminación de Datos

  • 🔒 Retención Local: Flash EEPROM Tool no almacena datos de usuario en servidores propios. Toda la información operativa reside en el almacenamiento local de tu dispositivo.
  • 🔒 Eliminación de Datos: Puedes eliminar completamente cualquier dato o caché generado por la aplicación desinstalándola de tu dispositivo o borrando los datos de la app desde la sección "Ajustes > Aplicaciones > Flash EEPROM Tool > Almacenamiento" de tu sistema operativo Android.
  • 🔒 Transparencia: El código fuente está disponible en GitHub para que cualquier usuario pueda auditar el manejo de datos.
